Algorithms

     Howdy, I'm Caleb Terry and this is a webpage I created to demostrate and explain many algorithms I find interesting. My goal is to not only showcase the most efficient methods but also the less optimal ones you can use to implement each particular algorithm.


     I decided to start this project because of a class I'm currently taking titled "Design and Analysis of Algorithms." In it we have so far covered a variety of problems which find their solution with a Dynamic, Recursive, Iterative, or Greedy approach. I have since become quite fasicinated with many of the algorithms we have gone over in this course. I wanted to implement them myself in the hopes of mastering the ability to translate the idea of the solution into acutal code which can be use to solve the problems.
